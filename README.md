# HR Attrition Analysis for Employee Retention

## 📌 Project Overview
This project aims to analyze employee data and uncover the main factors contributing to attrition. It includes EDA, ML modeling, SQL queries, and a Power BI dashboard.

## 🔧 Technologies Used
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- Scikit-Learn
- SQL
- Power BI

## 📊 Key Features
- Complete data preprocessing
- Exploratory Data Analysis with visualizations
- Random Forest ML Model
- Feature importance ranking
- HR insights & recommendations
- Interactive Power BI Dashboard

## 📁 Files Included
- `HR_Attrition_Analysis.ipynb`
- `powerbi_dashboard.pbix`
- `sql_queries.sql`
- `project_report.pdf`
- `README.md`

## 🧠 Insights
- Overtime and low salary significantly increase attrition.
- Job satisfaction and work-life balance are strong retention factors.
- Random Forest achieved ~85% prediction accuracy.
